Oii!

segue uma lista de comandos para fazer seu projeto funcionar

1° Intalar o venv 
    python -m venv venv
    - em seguida ative o ambiente virtual
    .\venv\Scripts\activate

2° Instalar o Django e o Crispy Forms
    pip install Django
    pip install django-crispy-forms

3° Criação do banco de dados 
    python manage.py makemigrations 
    python manage.py migrate 

4° Criação do login do admin 
    python manage.py createsuperuser

5° Rodando o projeto
    python manage.py runserver 

6° Teste o formulário cadastrando um algo e em seguida cheque se apareceu lá no admin
